# Random Password Generator

## Автор: Суворова Катя
## Описание:
Простое GUI-приложение для генерации случайных паролей с настройкой длины и типов символов. История паролей сохраняется в JSON.

## Требования:
- Python 3.11
- Библиотека tkinter (входит в стандартную поставку)

## Использование:
1. Запустите файл main.py.
2. Настройте параметры и нажмите "Сгенерировать".
3. История сохраняется автоматически.

## Примеры использования:
- Генерация пароля из 12 символов с цифрами и буквами.
- Генерация сложного пароля с использованием спецсимволов.#Random-Password-Generator
